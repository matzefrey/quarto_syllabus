# Ffm-syllabus Format

## Installing

Use the following to install with template file:

```bash
quarto use template tisprang/ffm-syllabus
```

Use the following to install just the extension:

```bash
quarto use template tisprang/ffm-syllabus
```

## Using

Adjust template file according to own purposes.
